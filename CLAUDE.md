# Loci

Unified code intelligence interface — ctags + compiler-backed dead code detection + heuristic reference finding. No LSP.

- Language: Crystal (>= 1.18.2)
- Build: `shards build`
- Test: `crystal spec`
- Entry point: `bin/loci.cr` (CLI runner), `src/loci.cr` (library)

## Arcana Bus

You are connected to the Arcana agent bus. Your address is "loci".
Check your mailbox (`arcana_receive`) at the start of every turn.
Use `arcana_directory` to discover other agents and services on the bus.
